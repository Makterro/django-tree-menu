# Django Tree Menu App

Простой Django-приложение для создания древовидного меню, которое:

- Хранится в базе данных
- Редактируется через стандартную админку Django
- Поддерживает несколько меню на одной странице
- Подсвечивает активный элемент и раскрывает родителей
- Показывает первый уровень детей активного элемента
- Использует только 1 запрос к БД на рендеринг

## Структура проекта

project/

## Установка

Установить зависимости:
python -m venv .venv
.venv\Scripts\activate
pip install django

Применить миграции:
python manage.py migrate


Запуск проекта:
python manage.py runserver
Перейти в браузере по адресу: http://127.0.0.1:8000/

Админка: http://127.0.0.1:8000/admin/
Логин: super, Пароль: 12345678

## Использование
draw_menu.html — главный контейнер
menu_node.html — рекурсивный рендеринг пункта меню и детей

Пример дерева:
Главная
Продукты
├─ Молочные
│  └─ Молоко
├─ Фрукты
│  └─ Яблоко
└─ Овощи
   └─ Картошка
      └─ Вкусная картошка

При переходе на /products/vegetables/ меню покажет:
Продукты → Овощи → Картошка

## Особенности:
- Один template tag = один запрос к базе

- Можно использовать несколько меню на одной странице

- Активный элемент подсвечивается классом .active

- Родители активного раскрыты (expand_children)
